## 🚀 Tecnologias

Esse projeto foi desenvolvido com as seguintes tecnologias:

### Backend:
- Java Spring e Spring MVC para criação do servidor
- Spring Data JPA para manipulação e persistência de dados
- Lombok
- Banco de dados PostgreSQL
### Frontend:
- React
- Typescript
- React Query
